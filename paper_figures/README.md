# paper_figures

Scripts to reproduce the tables and figures in the Trilobyte paper.

---

## Scripts

| File | Description |
|---|---|
| `trilobyte_table.py` | LaTeX table: Trilobyte vs FLAC vs LMIC baselines, sourced from WandB runs and result CSVs |
| `nac_compression_rate_comparisons.py` | Compression rate comparison plots across NAC-style compressors (FLAC, LDAC, LEC, etc.) |
| `sashimi_table.py` | LaTeX table for Sashimi model config comparison (BPB, EMA smoothing, compression rate) |

---

## Inputs

| Input | Source |
|---|---|
| `flac_eval_results.csv` | Generated by `baselines/flac_eval/flac_eval.py` |
| `lmic_eval_results.csv` | Generated by `baselines/in_context_eval/language_modeling_is_compression/compress_audio.py` |
| WandB runs | Logged during model training in `trilobyte/` |

---

## Usage

Each script can be run directly. Adjust WandB project/run IDs and CSV paths to match your environment.

```bash
python trilobyte_table.py
python nac_compression_rate_comparisons.py
python sashimi_table.py
```
